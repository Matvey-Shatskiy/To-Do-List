# 📝 Modern To-Do List Application

A full-stack task management application designed with a focus on productivity and clean UI. It allows users to track tasks, set priorities, manage deadlines, and archive completed items with a persistent SQLite backend.

------

## ✨ Features

- **Task Management:** Create, edit, and delete tasks with ease.
- **Priority System:** Categorize tasks by priority levels: Low, Medium, and High.
- **Archiving System:** Move completed or old tasks to an archive to keep your main workspace clean.
- **Upcoming Tasks:** A dedicated section that automatically highlights the next 3 most urgent tasks.
- **Persistent Storage:** All data is saved in a local SQLite database.
- **Modern UI/UX:** Built with Material UI, featuring glassmorphism effects, animated backgrounds, and a responsive layout.
- **Filtering:** Quickly switch between "Active" and "Archived" task views.

------

## 🛠️ Tech Stack

### Backend

- **Framework:** FastAPI (Python).
- **Database:** SQLite.
- **API:** RESTful endpoints for CRUD operations.

### Frontend

- **Framework:** React with TypeScript.
- **UI Library:** Material UI (MUI).
- **Styling:** Custom themes with backdrop filters and smooth animations.
